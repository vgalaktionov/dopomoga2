# dopomoga2

> New backend for dopomoga.nl

Built with:

-   Docker
-   Python
-   Django
-   Postgres
-   S3
-   Pico.css

## Setup

```bash
python3 -m venv venv
pip install -r requirements.txt

```

## Development

```bash
docker compose up --build --remove-orphans
```

## Deployment

On DigitalOcean App Platform:

```bash
# to main branch
git push
```
